In order to run the products please configure the following :
Go to File -> Project Structure -> Modules -> Signing Configs 
Choose key_temp then there will multiple row appear navigate to Store File and config the path that linked to key.jks according to your computer path.
take for instance my computer path to key.jks is /home/tuan/temp/ts/Fi-Note/src/key.jks.
For your computer the path may be differ so please re config the path before start running the project. The key.jks file is in the src folder.
